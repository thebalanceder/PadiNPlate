# Platform of Items Needed

Catalog of agricultural inputs and equipment with sourcing information.

## For Hackathon (MVP)

Simple list format with:
- Item name
- Estimated price
- Where to buy (local agrovet names)
- Contact information

## For Production (Full Product)

Integration options:
- Direct links to Shopee/TaniHub
- Agrove shop partnerships
- E-commerce API integration
- Mobile app marketplace
